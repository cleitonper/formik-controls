# Formik Controls
> Componentes ReactJS de formulário construidos com Formik

### Funcionalidades
  - Controle de estado
  - Validação de dados
  - Máscaras em formato de string

### Iniciando a aplicação

```bash
# Instale as dependências
yarn

# Execute o servidor de desenvolvimento
yarn start
```
